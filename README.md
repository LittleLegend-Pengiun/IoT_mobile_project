# IoT_mobile_project

### Note
Use react native expo

Require: [Nodejs LTS 16.18.0](https://nodejs.org/ko/blog/release/v16.18.0)

Install all dependacies
```
npm install
```

Run app on virtual device of Andriod studio
```
npm run android
```
